## The Release in One Sentence

v0.8 is best understood as an operator-quality release: tighter workflow primitives, better foundations for automation, and clearer extension paths for skills and integrations.

## What Matters Most

- Workflow composition is easier to reason about
- Skills and integration stories are stronger
- The release raises the value of automation and orchestration use cases

## What Operators Should Check

Before upgrading, check environment assumptions, connector compatibility, and any prompt or skill contracts you rely on in automation.

## Migration Checklist

- Snapshot configuration
- Re-run smoke tests for key skills
- Re-run cron jobs in a safe environment
- Verify messaging and MCP connectors
