## 什么是技能？

技能（Skills）是模块化插件，用于扩展 Hermes Agent 的内置能力。可以把它们理解为 AI Agent 的 App——每个技能教会 Hermes 一个新本领。

**内置技能**（预装）：
- `file-manager` — 文件读写和导航
- `web-search` — 网络搜索
- `code-runner` — 沙箱化代码执行

**社区技能**（由开源社区创建）：
- `github-integration` — 管理仓库、PR 和 Issue
- `web-scraper` — 从网页提取结构化数据
- `email-tool` — 发送和接收邮件
- `database-query` — 查询 SQL 和 NoSQL 数据库
- 还有 100+ 个技能...

## 浏览技能目录

### 在命令行中浏览

```bash
# 浏览所有可用技能
hermes skill search ""

# 按关键词搜索
hermes skill search "github"

# 按分类搜索
hermes skill search --category automation

# 查看技能详情
hermes skill info github-integration
```

```
📦 github-integration v2.1.0
   作者: @hermes-community
   下载量: 12,345
   评分: ⭐⭐⭐⭐⭐ (4.9/5)
   分类: 开发工具

   描述:
   完整的 GitHub 集成。创建仓库、管理 PR、审查代码、
   处理 Issue，自动化你的 GitHub 工作流。

   能力:
   - 创建和克隆仓库
   - 创建、审查和合并 Pull Request
   - 管理 Issue 和标签
   - 读写仓库中的文件
   - 触发 GitHub Actions

   前置要求:
   - GitHub Personal Access Token

   安装: hermes skill install github-integration
```

## 安装技能

### 基础安装

```bash
hermes skill install github-integration
```

```
📦 正在安装 github-integration v2.1.0...
✅ 安装成功！

📋 需要配置:
   该技能需要 GitHub Token 才能工作。
   运行: hermes skill configure github-integration
```

### 安装指定版本

```bash
hermes skill install github-integration@2.0.0
```

### 批量安装

```bash
hermes skill install github-integration email-tool web-scraper
```

### 配置技能

很多技能需要 API 密钥或配置：

```bash
hermes skill configure github-integration
```

```
🔧 配置 github-integration

GitHub Personal Access Token:
> ghp_xxxxxxxxxxxxxxxxxxxx

默认组织 (可选):
> my-company

默认分支:
> main

✅ 配置已保存！
```

## 管理已安装的技能

### 查看已安装技能

```bash
hermes skill list
```

```
已安装技能:
  ✅ file-manager    v1.0.0  (内置)
  ✅ web-search      v1.0.0  (内置)
  ✅ code-runner     v1.0.0  (内置)
  ✅ github-integration v2.1.0
  ✅ email-tool      v1.3.0
  ⏸️ web-scraper     v3.0.0  (已禁用)
```

### 启用/禁用技能

```bash
# 禁用（不卸载）
hermes skill disable web-scraper

# 重新启用
hermes skill enable web-scraper
```

### 更新技能

```bash
# 更新指定技能
hermes skill update github-integration

# 更新所有技能
hermes skill update --all
```

### 卸载技能

```bash
hermes skill remove email-tool
```

## 热门社区技能

| 技能 | 分类 | 描述 | 难度 |
|------|------|------|:----:|
| `github-integration` | 开发 | 完整 GitHub 工作流 | 🟢 |
| `web-scraper` | 数据 | 从任意网页提取数据 | 🟡 |
| `email-tool` | 通讯 | 通过 SMTP/IMAP 收发邮件 | 🟡 |
| `database-query` | 数据 | 查询 SQL 和 NoSQL 数据库 | 🟡 |
| `image-generator` | 创意 | 通过 DALL-E/SD 生成图片 | 🟢 |
| `pdf-reader` | 文档 | 提取 PDF 文本 | 🟢 |
| `docker-manager` | 运维 | 管理 Docker 容器 | 🟡 |
| `ssh-remote` | 运维 | 在远程服务器执行命令 | 🔴 |
| `wechat-notifier` | 通讯 | 发送企业微信通知 | 🟡 |
| `dingtalk-bot` | 通讯 | 钉钉机器人消息推送 | 🟡 |

> **国内用户推荐**：`wechat-notifier` 和 `dingtalk-bot` 是专为国内环境开发的技能，支持企业微信和钉钉的消息推送。

## 在对话中使用技能

安装完成后，技能自动可用。直接用自然语言提问就行：

```
> 创建一个新的 GitHub 仓库叫 "my-api"，用 Node.js 的 .gitignore

🔧 使用技能: github-integration
✅ 仓库已创建: https://github.com/yourname/my-api
   - 私有仓库
   - 已添加 Node.js .gitignore
   - 已初始化 README.md
```

### 技能链式调用

Hermes 可以在一个任务中组合多个技能：

```
> 查看我的 GitHub 通知，总结需要紧急处理的 PR，
> 然后用企业微信通知我

🔧 使用技能: github-integration, wechat-notifier
1. 检查 GitHub 通知... (5 条新通知)
2. 发现 2 个需要紧急审查的 PR
3. 生成摘要...
4. 发送企业微信通知...
✅ 通知已发送！
```

## 技能质量与安全

### 认证技能

带有 ✅ 标记的技能已经过社区团队审查：
- 安全性（无恶意代码）
- 质量（测试完善、文档齐全）
- 兼容性（适配最新 Hermes 版本）

### 权限管理

每个技能声明它需要的权限：

```bash
hermes skill permissions github-integration
```

```
github-integration 的权限:
  ✅ network    - 发送 HTTP 请求 (到 api.github.com)
  ✅ read-files - 读取本地文件
  ❌ write-files - 无文件写入权限
  ❌ execute    - 无 Shell 执行权限
```

安装前务必检查权限，特别是未认证的技能。
