## 为什么选择 Discord + Hermes？

Discord 是开发者社区的主要聚集地。将 Hermes Agent 添加到你的 Discord 服务器，可以把它变成一个 AI 驱动的知识中心：

- **斜杠命令** — `/ask`、`/code`、`/explain` 快速交互
- **线程支持** — 长对话放在线程里，频道保持整洁
- **角色权限** — 控制谁能使用机器人
- **富文本** — 代码块、嵌入卡片和交互按钮
- **多频道** — 不同频道配置不同用途

## 准备工作

- [安装并配置好 Hermes Agent](/zh/tutorials/getting-started/installation)
- 拥有 Discord 账号并有服务器管理权限
- Node.js 18+ 已安装
- [AI 模型已配置](/zh/tutorials/getting-started/choosing-a-model)

## 第 1 步：创建 Discord 应用

### 注册机器人

1. 访问 [Discord Developer Portal](https://discord.com/developers/applications)
2. 点击 **New Application**，命名为 "Hermes AI"
3. 在侧栏进入 **Bot** 页面
4. 点击 **Add Bot** → **Yes, do it!**
5. 在 **Token** 下点击 **Copy** 复制 Bot Token

> **安全提示**：绝不分享或提交这个 Token。

### 设置权限

在 **Bot** 页面：

- 开启 **Message Content Intent**（读取消息必需）
- 开启 **Server Members Intent**（角色权限必需）

### 生成邀请链接

进入 **OAuth2 → URL Generator**：

1. 勾选 Scopes：`bot`、`applications.commands`
2. 勾选权限：
   - Send Messages（发送消息）
   - Send Messages in Threads（在线程中发送）
   - Create Public Threads（创建公开线程）
   - Embed Links（嵌入链接）
   - Attach Files（附件）
   - Read Message History（读取历史）
   - Use Slash Commands（使用斜杠命令）

3. 复制生成的 URL，打开它邀请机器人到你的服务器

## 第 2 步：配置 Hermes

```bash
# 设置 Discord Bot Token
hermes config set discord.bot_token "MTIzNDU2Nzg5MDEy...你的token"
hermes config set discord.enabled true

# 限制到特定频道（推荐）
hermes config set discord.allowed_channels "[频道ID]"
```

### 完整 Discord 配置

```yaml
# ~/.hermes/config.yaml
discord:
  enabled: true
  bot_token: "MTIzNDU2Nzg5MDEy..."

  # 访问控制
  allowed_channels: [1234567890123456789]
  allowed_roles: ["AI-User", "Developer"]
  admin_roles: ["Admin", "Moderator"]

  # 行为配置
  respond_to_mentions: true      # @提及时回复
  respond_to_dms: true           # 回复私信
  use_threads: true              # 用线程组织长对话
  thread_auto_archive: 1440      # 线程 24 小时后自动归档

  # 速率限制
  max_requests_per_user: 30      # 每小时
  cooldown_seconds: 5            # 消息间隔

  # 外观
  embed_color: "#d97706"         # Hermes 品牌色
  show_model_info: true          # 显示当前使用的模型
```

## 第 3 步：注册斜杠命令

Hermes 启动时会自动注册斜杠命令：

| 命令 | 说明 |
|------|------|
| `/ask <问题>` | 向 Hermes 提问 |
| `/code <语言> <描述>` | 生成代码 |
| `/explain <代码>` | 解释代码或概念 |
| `/review` | 审查代码（附文件） |
| `/model <名称>` | 切换 AI 模型 |
| `/session <操作>` | 管理会话 |
| `/help` | 显示帮助 |

## 第 5 步：在 Discord 中使用

### 斜杠命令示例

```
/ask 如何在 Express 中实现 JWT 认证？

/code python 创建一个爬取知乎热榜的脚本

/explain ```
const memoize = (fn) => {
  const cache = new Map();
  return (...args) => {
    const key = JSON.stringify(args);
    if (cache.has(key)) return cache.get(key);
    const result = fn(...args);
    cache.set(key, result);
    return result;
  };
};
```
```

### @提及模式

开启 `respond_to_mentions` 后：

```
@Hermes AI 帮我优化这个 SQL 查询
```

Hermes 会在线程中回复，保持频道整洁。

### 线程对话

启用线程后，第一次回复会创建线程，后续对话在线程中进行：

```
频道: #开发求助
├── 用户: @Hermes AI 帮我把应用部署到 Docker
│   └── [线程: 部署到 Docker]
│       ├── Hermes: 好的！我来帮你设置 Docker 部署...
│       ├── 用户: 我要用多阶段构建
│       ├── Hermes: 好选择！多阶段构建的 Dockerfile...
│       └── 用户: 怎么配 Nginx 反代？
```

## 高级功能

### 按角色分配模型

不同角色使用不同级别的模型：

```yaml
discord:
  role_models:
    "Free": "openai/gpt-4o-mini"              # 入门模型
    "Developer": "anthropic/claude-3.5-sonnet"  # 完整权限
    "Admin": "anthropic/claude-3.5-sonnet"      # 完整权限 + 设置管理
```

### 自定义斜杠命令

```yaml
discord:
  custom_commands:
    - name: "translate"
      description: "翻译文本"
      template: "将以下内容翻译成{language}: {text}"
      parameters:
        - name: "text"
          type: "string"
          required: true
        - name: "language"
          type: "string"
          required: true
          choices: ["英文", "中文", "日文", "韩文"]

    - name: "summarize"
      description: "总结内容"
      template: "简洁地总结: {input}"
```

### 按频道配置不同人设

```yaml
discord:
  channel_contexts:
    1234567890:  # #编程求助
      system_prompt: "你是一个编程助手，总是提供代码示例。用中文回复。"
      model: "anthropic/claude-3.5-sonnet"
    9876543210:  # #闲聊
      system_prompt: "你是友好的 AI 助手，保持简短回复。用中文回复。"
      model: "openai/gpt-4o-mini"
```

## 常见问题

### 机器人显示离线

```bash
pm2 status hermes-discord
hermes discord test
```

### 斜杠命令不出现

全局斜杠命令最长需要 1 小时传播。立即测试用服务器级命令：

```bash
hermes discord register-commands --guild 服务器ID
```
