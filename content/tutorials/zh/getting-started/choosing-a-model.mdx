## 为什么模型选择很重要？

Hermes Agent 与模型无关——它通过 OpenRouter、OpenAI、Anthropic 等提供商支持 200+ 种模型。但不同模型差异巨大，你的选择直接影响：

- **回答质量** — 输出的准确性和有用程度
- **响应速度** — Hermes 回复的快慢
- **使用成本** — 每次对话花多少钱
- **能力范围** — 能否处理代码、图片、长文档等

选模型就像选工具——你不会用大锤去敲图钉，也不会用小模型做复杂的多文件重构。

## 模型分级详解

### 第一梯队：旗舰模型（最高质量）

这是目前能力最强的模型。适合需要深度推理、多步规划或高质量代码生成的复杂任务。

| 模型 | 提供商 | 优势 | 费用（每百万 tokens） |
|------|--------|------|---------------------|
| Claude 3.5 Sonnet | Anthropic | 代码生成、分析、长上下文 | ~$3 输入 / $15 输出 |
| GPT-4o | OpenAI | 多模态、通用、稳定 | ~$2.50 输入 / $10 输出 |
| Gemini 1.5 Pro | Google | 百万 token 上下文、多模态 | ~$1.25 输入 / $5 输出 |

```bash
# 设置旗舰模型
hermes --model anthropic/claude-3.5-sonnet
```

> **适用场景**：架构设计、复杂调试、多文件重构、技术写作、安全审计。

### 第二梯队：性价比模型（日常首选）

质量优秀，价格仅为旗舰模型的几分之一，非常适合日常使用。

| 模型 | 提供商 | 优势 | 费用（每百万 tokens） |
|------|--------|------|---------------------|
| Claude 3 Haiku | Anthropic | 快速、便宜、擅长代码 | ~$0.25 输入 / $1.25 输出 |
| GPT-4o Mini | OpenAI | 极速、实惠、可靠 | ~$0.15 输入 / $0.6 输出 |
| Llama 3.1 70B | Meta (OpenRouter) | 开源、无数据共享 | ~$0.50 输入 / $0.75 输出 |

```bash
# 日常使用性价比模型
hermes --model openai/gpt-4o-mini
```

> **适用场景**：日常编码、文件操作、简单自动化、快速问答。

### 第三梯队：本地模型（免费 + 隐私）

完全在自己的硬件上运行。零成本、最高隐私，但需要一块不错的显卡。

| 模型 | 参数量 | 显存需求 | 质量评价 |
|------|--------|----------|----------|
| Llama 3.1 8B | 80 亿 | 8GB | 简单任务足够 |
| Mistral 7B | 70 亿 | 6GB | 快速、通用 |
| CodeLlama 34B | 340 亿 | 24GB | 代码能力强 |
| Llama 3.1 70B | 700 亿 | 48GB+ | 接近旗舰质量 |
| Qwen2 72B | 720 亿 | 48GB+ | 中文能力最强 |

```bash
# 使用 Ollama 运行本地模型
hermes --model ollama/llama3.1:8b --api-base http://localhost:11434
```

> **国内推荐**：如果你主要用中文，强烈推荐通义千问 Qwen2 系列。72B 版中文能力在开源模型中排名第一，7B 版在消费级显卡上也能流畅运行。

## 配置 OpenRouter（推荐）

[OpenRouter](https://openrouter.ai) 是推荐的提供商——一个 API Key 就能访问 200+ 模型。

### 第 1 步：获取 API Key

1. 访问 [openrouter.ai](https://openrouter.ai) 注册账号
2. 进入 **Settings → API Keys**
3. 点击 **Create Key**，复制密钥

> **付费方式**：OpenRouter 支持信用卡和加密货币充值。如果没有境外信用卡，可以考虑使用虚拟信用卡服务（如 WildCard）。

### 第 2 步：配置 Hermes

```bash
hermes config set api_key sk-or-v1-your-key-here
hermes config set default_model anthropic/claude-3.5-sonnet
```

### 第 3 步：验证连接

```bash
hermes --test-connection
# ✅ 已连接到 OpenRouter
# 模型: anthropic/claude-3.5-sonnet
# 余额: $12.50
```

## 实时切换模型

Hermes 支持在对话中随时切换模型：

```
> /model openai/gpt-4o-mini
✅ 已切换到: gpt-4o-mini

> 总结一下这个日志文件
（使用 gpt-4o-mini — 又快又便宜）

> /model anthropic/claude-3.5-sonnet
✅ 已切换到: claude-3.5-sonnet

> 分析错误模式并给出修复建议
（使用 Claude — 更擅长复杂分析）
```

### 创建模型别名

为常用模型设置快捷方式：

```yaml
# ~/.hermes/config.yaml
model_aliases:
  快: openai/gpt-4o-mini
  强: anthropic/claude-3.5-sonnet
  码: anthropic/claude-3.5-sonnet
  本地: ollama/qwen2:7b
```

```
> /model 快
> /model 强
> /model 本地
```

## 模型选择决策树

```
这个任务复杂吗？（多文件、架构设计、调试）
├── 是 → 用 Claude 3.5 Sonnet 或 GPT-4o
└── 否
    ├── 在意成本吗？
    │   ├── 是 → 用 GPT-4o Mini 或 Claude Haiku
    │   └── 否 → 用 Claude 3.5 Sonnet
    └── 需要隐私保护吗？
        ├── 是 → 用本地模型（Llama 3.1 或 Qwen2）
        └── 否 → 用 GPT-4o Mini
```

## 省钱策略

### "逐级升级"法

从便宜模型开始，质量不满意再升级：

1. **第一次尝试**：GPT-4o Mini（~$0.15/百万 tokens）
2. **不满意时**：切换到 Claude 3.5 Sonnet（~$3/百万 tokens）
3. **关键任务**：使用 GPT-4o 或 Claude + 延伸思考模式

这种方法可以比全程使用旗舰模型**节省 60-80% 的费用**。

### 月费预估

| 使用强度 | 全用便宜模型 | 混合策略 | 全用旗舰模型 |
|---------|:-:|:-:|:-:|
| 轻度（50 条/天） | ~¥15/月 | ~¥60/月 | ~¥180/月 |
| 中度（200 条/天） | ~¥60/月 | ~¥220/月 | ~¥720/月 |
| 重度（500+ 条/天） | ~¥150/月 | ~¥540/月 | ~¥1800/月 |

## 配置本地模型

想要最高隐私和零持续费用，就在本地运行模型。

### 使用 Ollama

```bash
# 安装 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 下载模型（推荐：中文首选 Qwen2）
ollama pull qwen2:7b

# 或者下载 Llama 3.1
ollama pull llama3.1:8b

# 配置 Hermes
hermes config set provider ollama
hermes config set api_base http://localhost:11434
hermes config set default_model qwen2:7b
```

### 使用 LM Studio（适合 Windows 用户）

1. 下载 [LM Studio](https://lmstudio.ai)
2. 搜索并下载模型（推荐：Qwen2 7B Instruct）
3. 在 LM Studio 中启动本地服务器
4. 配置 Hermes：

```bash
hermes config set provider openai-compatible
hermes config set api_base http://localhost:1234/v1
hermes config set default_model local-model
```

> **Windows 用户注意**：需要在 WSL2 中运行 Hermes。LM Studio 可以在 Windows 本机运行，Hermes 从 WSL2 中通过 `http://localhost:1234` 访问。
