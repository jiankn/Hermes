## 环境要求

在安装 Hermes Agent 之前，请确保你的系统满足以下要求：

- **操作系统**：Linux（Ubuntu 20.04+、Debian 11+）、macOS 12+、WSL2 或 Android Termux
- **Node.js**：v18 或更高版本
- **Git**：最新版本
- **AI API 密钥**：OpenRouter、OpenAI、Anthropic 或任何 OpenAI 兼容提供商

> **提示**：还没有 API 密钥？推荐使用 [OpenRouter](https://openrouter.ai) —— 一个 key 就能访问 200+ 模型。国内用户可以考虑使用代理或者选择国产 API 提供商。

## 极速安装（推荐）

最快的安装方式：

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/install.sh | bash
```

这个脚本会自动完成：
1. 检查系统要求
2. 全局安装 Hermes Agent
3. 创建默认配置
4. 运行初始化向导

安装完成后，验证是否成功：

```bash
hermes --version
# 预期输出: hermes-agent v0.8.x
```

> **国内用户注意**：如果下载速度很慢，可以配置 HTTP 代理：
> ```bash
> export https_proxy=http://127.0.0.1:7890
> ```

## 手动安装

如果你更喜欢手动安装，或者快速脚本不起作用：

### 第 1 步：克隆仓库

```bash
git clone https://github.com/NousResearch/hermes-agent.git
cd hermes-agent
```

### 第 2 步：安装依赖

```bash
npm install
```

使用国内 npm 镜像加速（可选）：

```bash
npm install --registry=https://registry.npmmirror.com
```

### 第 3 步：配置 API 密钥

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入你的 API 密钥：

```env
OPENROUTER_API_KEY=sk-your-api-key-here
DEFAULT_MODEL=anthropic/claude-3.5-sonnet
```

### 第 4 步：启动 Hermes

```bash
npm start
```

你应该看到：

```
🚀 Hermes Agent is active and ready.
Type your message or use /help for commands.
```

## 各平台特别说明

### macOS

在 macOS 上，你可能需要先安装 Xcode 命令行工具：

```bash
xcode-select --install
```

### WSL2（Windows 用户）

确保使用 WSL2，而不是 WSL1：

```bash
wsl --set-default-version 2
```

然后在 WSL2 终端中按照 Linux 步骤操作。

### Android Termux

```bash
pkg install nodejs git
```

然后按照手动安装步骤操作。

## 常见问题排查

### 错误："Permission denied"

```bash
chmod +x install.sh
sudo ./install.sh
```

### 错误："Node.js 版本太旧"

使用 nvm 更新 Node.js：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 18
nvm use 18
```
